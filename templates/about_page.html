About us

Names. They’re important.
Connect with confidence.

Getting someone’s name right is not only respectful, it’s essential. NOMI helps you to identify how a person's name is pronounced correctly.
NOMI allows users to create and share a name profile, which shows the user’s name, pronunciation, their pronouns, and name etymology. This is how we make names accessible to all. We champion diversity by normalising name familiarity across cultures.
